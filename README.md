https://bilgihankose.github.io/twitter-tailwind-clone

## Run Locally

Clone the project

```bash
  git clone https://github.com/bilgihankose/twitter-tailwind-clone.git
```

Go to the project directory

```bash
  cd twitter-tailwind-clone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
